Nama saya Silva Oktaria Putri, mahasiswa semester 5 Program Studi Teknik Informatika di Institut Teknologi Sumatera. Saya memiliki minat besar dalam dunia teknologi dan pendidikan. Selama kuliah, saya aktif sebagai asisten praktikum untuk mata kuliah Pengantar Komputer dan Software. Dalam peran ini, saya bertanggung jawab mempersiapkan materi, membantu mahasiswa memahami konsep, serta menilai tugas dan ujian mereka. Saya menguasai berbagai bahasa pemrograman seperti Python dan C++, serta mahir menggunakan software seperti Microsoft Office, Adobe Photoshop, dan MATLAB. Saya juga memiliki keterampilan dalam analisis data. Saya sangat tertarik untuk melanjutkan kontribusi saya sebagai asisten praktikum dan bersemangat untuk membantu mahasiswa lebih memahami dan menguasai perangkat komputer dan software.
